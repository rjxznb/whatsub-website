# @whatsub/website

Marketing + download landing page for whatSub. Next.js 14 + Tailwind +
static export. Deploys to /data/whatsub-web/ on the Eversay Aliyun ECS,
served by enghub's nginx at https://whatsub.eversay.cc/.

See ../docs/superpowers/specs/2026-05-09-whatsub-landing-page-design.md
for the full design spec, and the sibling plan file for the implementation
plan this codebase executes.
